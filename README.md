# Tina Uma — Portfolio Site

Personal portfolio of **Tina Umasheva** — Prompt Engineer / Vibe-coder / Product Designer.

**Live site:** https://tinauma.github.io/ai-vibe-station/

## Stack

- [Astro](https://astro.build/) — static site generator
- Tailwind CSS v4
- Deployed via GitHub Actions → GitHub Pages

## Local development

```sh
npm install
npm run dev       # http://127.0.0.1:4321
npm run build     # build to ./dist
```
